# 요청사항 요약
- **사용자 질문**: Git에 설정된 현재 유저(이름, 이메일)를 조회하는 기능 및 명령어 문의.

# 작업내용 및 응답
터미널에서 사용자가 여러 명령어를 시도해 보신 내역(`git author`, `git user.name`)을 확인했습니다. 아쉽게도 해당 명령어들은 존재하지 않지만, 거의 근접하게 접근하셨습니다!

Git에서 등록된 사용자 이름과 이메일을 조회하는 정확한 명령어는 다음과 같습니다. 터미널 창에 그대로 입력하시면 됩니다.

### 1. 사용자 이름 조회
```bash
git config user.name
```

### 2. 사용자 이메일 조회
```bash
git config user.email
```

**[현재 상태 점검]**
제가 현재 사용자님의 터미널에서 위 명령어들을 백그라운드로 조회해 본 결과, **아무런 값도 출력되지 않았습니다.** 
이는 즉, **"아직 사용자님의 맥에 Git 이름과 이메일이 등록되지 않은 상태"**임을 의미합니다. (따라서 에러가 나거나 아무것도 안 뜨는 것이 정상입니다.)

### 💡 어떻게 해야 할까요?
Git을 이용해 버전을 관리하고 커밋(저장)을 하려면 누가 작업했는지 기록하기 위해 최초 1회 유저 등록이 반드시 필요합니다. 터미널에 아래 명령어를 입력하여 등록해 주세요!

```bash
# 본인의 이름이나 닉네임을 따옴표 안에 넣어 입력하세요.
git config --global user.name "내이름"

# 본인의 이메일(가급적 Github 가입 이메일)을 따옴표 안에 넣어 입력하세요.
git config --global user.email "내이메일@example.com"
```

입력하신 후에 다시 `git config user.name` 과 `git config user.email` 을 입력해 보시면, 방금 등록한 이름과 이메일이 터미널에 예쁘게 출력되는 것을 확인하실 수 있습니다.
